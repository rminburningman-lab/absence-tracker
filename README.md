# Memory Timeline / Absence Tracker

A premium, emotional, and cinematic mobile experience for reflecting on the passage of time.

## Overview

Memory Timeline / Absence Tracker is a concept-driven mobile application designed to turn time into an intimate experience rather than a mere countdown. It combines ideas from time tracking, journaling, calendars, memory archives, emotional analytics, and personal storytelling into a calm and elegant interface.

The product is built around a simple philosophy:

> Time should not only be counted — it should be experienced.

## Core Experience

The app aims to help users feel the passage of time through multiple perspectives:

- Chronological time
- Emotional time
- Seasonal time
- Astronomical time
- Personal memory time

## Current Status

The project currently includes:

- A React Native + Expo foundation
- A polished dark-mode prototype for the home experience

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Install dependencies

```bash
cd app
npm install
```

### Run the app

```bash
cd app
npm start
```
